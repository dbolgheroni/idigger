Brazilian IBrA index Greenblatt Table

This is the Greenblatt method applied to the brazilian IBrA index stocks. The table lists the stocks ordered by the Greenblatt method, and the highlighted ones represent the buying recommendation of today. The reference is used to calculated the gain based on the recommended stocks of this specific date. You can change it and click Generate to use another reference. The software will do the calculations based on what is in the database until now.

The software runs everyday except on weekends, gather data and process it accordingly. A webservice with a RESTful API provides the data in which this site uses. The API is under development, not documented, not stable yet. The code webpage is available here.

If you don't know how the Greenblatt method works, do your own research. This software is experimental, use at your own risk.
